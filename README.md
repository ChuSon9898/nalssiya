# ☑️ 주요 기능

### ⭐ 1. 현 위치 일기예보
- 현 위치의 시간별 및 주간 일기예보
- 그 외에 풍속, 습도 등 다양한 날씨 정보



### ⭐ 2. 지역 검색 및 검색 지역 일기예보
- 시, 구 별로 지역 검색
- 즐겨찾는 지역의 시간별 및 주간 일기예보
- 그 외에 풍속, 습도 등 다양한 날씨 정보


### ⭐ 3. 지역 북마크 기능
- 원하는 지역의 북마크 추가 및 삭제 기능
- 북마크 추가된 모든 지역은 간단한 날씨 정보 확인 가능


### ⭐ 4. 날씨 알림 기능
- 위치 정보를 기반으로 매일 오전 당일의 대략적인 날씨 알림 제공
